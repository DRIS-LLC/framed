## STRAIGHT FROM Y2K.

A 2000s-style component library refreshed for the 21st century.

_Inspired by the Fruitger Aero aesthetic, early iPhone UI, & [Jordan Singer](https://x.com/jsngr)._

### INSTALLATION

The base installation for these components are the same as [shadcn/ui](https://ui.shadcn.com/docs/installation).

To get started, add the following config files to your project:

### BUTTON

<ComponentContainer>
  {config.VARIANTS.map((variant) => (
    <Button key={variant} variant={variant}>
      {variant}
    </Button>
  ))}
</ComponentContainer>

#### CODE

<Code>pnpm dlx shadcn@latest add button</Code>
